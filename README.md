# GhostTrack‑v2 — Sistema Reale Unificato

GhostTrack‑v2 è l’ecosistema ufficiale che integra in un’unica architettura:

- UI reale basata su pannelli HTML statici
- Civil Defender Wallet (modulo ufficiale)
- RZA‑pulciz (motore cognitivo modulare)
- Quantum Layer
- ZDOS13 HyperOS
- Aurora Protocol
- Libreria documentale completa
- Pipeline di build unificata

Questo repository rappresenta la versione **reale, attiva e funzionante** del sistema GhostTrack.  
Tutto ciò che era presente nel repository *GhostTrack‑v2‑edu* è stato importato, convertito e integrato in questo ecosistema.

---

## Struttura del Sistema

- `docs/` — Documentazione ufficiale + pannelli UI
- `webapp/static/` — UI compilata
- `modules/` — Moduli core (wallet, RZA, quantum, ZDOS13)
- `rza_pulciz/` — Motore cognitivo
- `quantum/` — Quantum Engine + manifest
- `wiki/` — Libreria tecnica estesa
- `var/` — Stato ecosistema
- `api/` — Endpoint RZA
- `cli/` — Strumenti da linea di comando
- `build/` — Artefatti di build
- `logs/` — Log di sistema

---

## EDU Imported Modules

Il repository *GhostTrack‑v2‑edu* è stato completamente integrato.  
Tutti i pannelli, script, documenti e moduli sono ora parte del sistema reale.
